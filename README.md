Цей проект написаний на мові Python і виконує обчислення середнього значення (Математичного очікування) та стандартного відхилення для заданих числових значень.

Інструкція з встановлення та використання
Встановіть Python на ваш комп'ютер, якщо він ще не встановлений. Інструкції з встановлення Python можна знайти на офіційному сайті Python (https://www.python.org).

Завантажте файл з кодом main.py з цього репозиторію на ваш комп'ютер.

Відкрийте командний рядок або термінал на вашому комп'ютері.

Перейдіть до папки, в яку ви завантажили файл main.py за допомогою команди cd.

Запустіть програму за допомогою команди python main.py.

Програма буде запитувати вас ввести числові значення для кожного завдання. Введіть значення, а потім натисніть Enter.

Продовжуйте вводити числові значення для кожного завдання. Якщо ви бажаєте завершити введення, введіть 0 і натисніть Enter.

Після завершення введення програма виведе довірчий інтервал 95% для проекту.

Вимоги
Цей проект вимагає встановлення Python версії 3.x.
